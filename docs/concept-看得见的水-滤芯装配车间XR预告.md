## 二、立意来源:三个真实功能模块

本片每一拍都对应产品里一个真实模块(代码已核实):

- **点检** `features/inspection/checklist-panel.js`
  任务清单(已完成/紧急项)→ 沿引导走到热点 → 看实测数据 + 历史趋势 → 通过 / 异常 / 拍照。
- **监控** `core/app-state.js` 中的两类热点:`monitorVideo`(监控视频)+ `monitorData`(实时数据)
  把摄像头画面与实时数据**锚定在全景里**,异常变红。(代码标记 `待完善`,正是"未来"该长成的样子。)
- **学习 / 教学** `features/narration/*` + `features/study`
  分步语音讲解 → 同步高亮 ROI 区域 → 结尾**测验**确认掌握;"禅定模式"嵌入课程。

> 形态进化:**今天用一块平板看 2D 全景 → 未来戴 XR 眼镜,数字孪生叠加在真实车间上。**

## 三、串联设计:一条产线的故事(真实膜材料车间)

场景锚定在你们真实的**膜元件生产**洁净车间,工段:**卷制 → 组装/灌胶 → 通量测试**。
用一条暗线(卷制张力)把三个功能缝成闭环:

**(点检)发现卷制张力上行 →(监控)盯住全线 → 张力变红、呼出老师傅分身 →(学习)老师傅正好传授卷制手艺 → 通量测试见清水。**
三个功能,一条产线,同一个无形 AI「澄」+ 一位随叫随到的老师傅全息分身。台词与画面逐镜对齐。

## 六、未来车间美术设定(以真实照片为基因,向前推一代)

保留可辨识 DNA:灰色镜面环氧地坪 · 黄色导引线 · 头顶黑色桁架 · 中央滤芯机组 · 洁净白基调 · 无尘服。

| 元素 | 现状(照片) | 片中 · 未来升级 |
|---|---|---|
| 空间 | 实墙、管线外露、照明偏平 | 更通透,局部玻璃 / 亚克力隔断,层次更深 |
| 物理屏幕 | 有控制屏 / 仪表 | **几乎为零**;信息全部移入 XR 眼镜 |
| 光 | 顶灯硬光、反光杂乱 | 均匀柔和的环境光,克制自发光;靠材质与留白显高级 |
| 色彩 | 工业杂色 | 物理空间近乎无彩(灰白)→ 颜色(蓝 / 绿 / 橙 / 红)**只在 XR 信息层**出现,色彩即信息 |
| 地面黄线 | 喷涂静态线 | 嵌入式发光导引带,XR 下顺势"长"成光路 |
| 装配机组 | 金属机架 + 滤芯 | 更精密、有雕塑感,半透外壳为"显形"留接口 |

**分寸**:升级是为"更好的明天"的信服感,不是赛博朋克。底子仍是克制、洁净、专业的工业未来——干净比酷更重要。

## 七、视觉语言(全片统一)

- **沿用产品设计 Token 作为 XR 的 HUD 语言**:品牌蓝 `#4a8eff` 主色 + 绿(正常)/ 橙(警告)/ 红(报警)语义色 + 等宽字体显数据 + Lucide 细描边图标。真实产品与幻想片的连续性由此建立。
- **三个视觉母题**:① 水即光(水流 / 数据流皆为流动的光);② 显形(看不见的参数、趋势、机器状态被看见);③ 黄线 → 光路(真实地面线延伸成 XR 引导路径,锚定"真实↔幻想")。
- **声音**:极轻的车间底噪 + 澄克制的合成嗓音 + 一个标志性"显形"音效;学习一拍音乐转暖。
